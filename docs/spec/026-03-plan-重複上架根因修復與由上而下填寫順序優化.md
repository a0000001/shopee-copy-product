## 二、 問題診斷與精確破口分析

### 1. 重複上架的確切根因 (Retry Status Misjudgment)
- **現象**：商品實際上已經在蝦皮後台成功建立上架，但 `batch-upload.js` 的日誌卻顯示失敗，隨後引發 `processItemWithRetry` 進行第 2 次重試上傳，導致同一個商品被重複刊登兩次。
- **根因分析**：
  - 在點擊「儲存並上架」按鈕後，蝦皮賣家中心成功提交並跳轉 URL 至商品列表 (`/portal/product/list`)。
  - 當跳轉發生時，舊 Tab 的 Content Script 隨即銷毀。`clickSaveButton` 或其後續的狀態查詢收到 Chrome 訊息連線中斷（`Connection closed` 或未及時回傳 `{ ok: true }`），被 `batch-upload.js` 外層 `catch` 當作失敗處理。
  - 由於被誤判為失敗，外層觸發了 `await processItemWithRetry(item)` 的重試邏輯，再次開啟新 Tab 重新刊登。

### 2. 由上而下欄位填寫順序優化
- 原先 `fillAll()` 的順序未完全貼合蝦皮 UI Layout，且填寫了預設即有的欄位，造成 Validation 提示短暫閃爍。
- 新順序完全對齊蝦皮新增商品頁由上至下結構：
  1. **商品名稱** (`name`)
  2. **類別** (`category`)
  3. **屬性 -> 品牌** (`brand`)
  4. **屬性 -> 包裝尺寸** (`dimension`)
  5. **商品描述** (`description`)
  6. **價格** (`price`)
  7. **數量** (`stock`)
  8. **最低數量**（**跳過不填**，保持蝦皮預設值 `1`）
  9. **運費 -> 重量**（**跳過不填**，保持蝦皮預設空白）
  10. **信用卡分期** (`installment`)

## 三、 擬定修復方案 (Proposed Changes)

### Component 1: `batch-upload.js` 上架成功與跳轉之精確判定

#### [MODIFY] [batch-upload.js](file:///S:/projects/shopee-copy-product/extension/batch-upload.js)
- 在 `clickSaveButton` 發送點擊後，監聽分頁 URL 是否變更為 `/portal/product/list` 或跳轉至商品列表。
- 若偵測到 URL 跳轉至商品列表頁，即代表蝦皮後台已成功接收並創建商品，**強制標記為成功 (`success = true`)**，禁止進入重試流程。

```javascript
// 修改 extension/batch-upload.js fillAndSaveSingle 第三段按鈕點擊
const clickRes = await chrome.tabs.sendMessage(tabId, { action: 'clickSaveButton' }).catch(e => null)

// 點擊後等待 2 秒檢查 Tab 網址，若已跳轉至商品列表頁，視為成功上架
await sleep(2000)
const finalTab = await chrome.tabs.get(tabId).catch(() => null)
if (finalTab && (finalTab.url.includes('/portal/product/list') || !finalTab.url.includes('/portal/product/new'))) {
  console.log('[SGC] Tab redirected to list page, upload confirmed successful')
  return true
}

if (clickRes && clickRes.ok) return true
```

### Component 2: `content.js` `fillAll()` 由上而下順序重構

#### [MODIFY] [content.js](file:///S:/projects/shopee-copy-product/extension/content.js)
- 重構 `fillAll()` 的執行順序，並移除「最低數量」與「重量」的賦值填寫：

```javascript
async function fillAll(data) {
  const results = []

  // 1. 商品名稱
  const title = data.ps_product_name || data.title || ''
  results.push({ field: '商品名稱', ...(await fillFieldAsync(title, () => findFieldByLabel('商品名稱'), '[data-product-edit-field-unique-id="name"] input.eds-input__input')) })

  // 2. 類別
  try {
    const catResult = await fillCategoryAsync(data)
    results.push({ field: '類別', ...catResult })
    if (catResult.ok && !catResult.alreadySelected) await new Promise(r => setTimeout(r, 1000))
  } catch (e) { results.push({ field: '類別', ok: false, error: e.message }) }

  // 3. 屬性 -> 品牌
  const brandName = data.ps_brand || 'NoBrand'
  try {
    const brandResult = await fillBrandAsync(brandName)
    results.push({ field: '品牌', ...brandResult })
  } catch (e) { results.push({ field: '品牌', ok: false, error: e.message }) }

  // 4. 屬性 -> 包裝尺寸
  const dimStr = data.dimension || (data.ps_length && data.ps_width && data.ps_height ? `${data.ps_length}x${data.ps_width}x${data.ps_height}` : '')
  if (dimStr) {
    results.push({ field: '尺寸（長 x 寬 x 高）', ...(await fillFieldAsync(dimStr, () => findFieldByLabel('尺寸（長 x 寬 x 高）'))) })
  }

  // 5. 商品描述
  const desc = data.ps_product_description || data.description || ''
  if (desc) {
    results.push({ field: '商品描述', ...(await fillFieldAsync(desc, () => findFieldByLabel('商品描述'), '[data-product-edit-field-unique-id="description"] .ql-editor')) })
  }

  // 6. 價格
  const price = data.ps_price ?? data.price ?? ''
  let cleanPrice = price != null && price !== '' ? String(price).split('~')[0].split('-')[0].replace(/[^\d]/g, '').trim() : ''
  if (cleanPrice) {
    const priceRes = await fillFieldAsync(cleanPrice, () => findFieldByLabel('價格'), '[data-product-edit-field-unique-id="price"] input.eds-input__input')
    results.push({ field: '價格', ...priceRes })
    if (priceRes.ok) await new Promise(r => setTimeout(r, 600))
  }

  // 7. 數量
  const stockVal = data.ps_stock != null ? String(data.ps_stock) : '999'
  results.push({ field: '商品數量', ...(await fillFieldAsync(stockVal, '[data-product-edit-field-unique-id="stock"] input.eds-input__input:not([type="file"])')) })

  // 8. 最低數量：不用填，保持預設值 1
  // 9. 重量：不用填，保持預設空白

  // 10. 信用卡分期
  const installment = data.installment
  if (installment) { /* 信用卡分期處理... */ }

  return { ok: true, results }
}
```
