# 商品描述-spec
- 商品描述若有設備規格需去除, 並把設備規格放到device_specs字段,
- 商品描述沒有提到設備規格 就由你根據商品描述來推斷，補充設備規格

設備規格/computer_specs 禁止偷懶。第一步：到 S:\projects\ai-hardware-checker\models.json 查該模型。第二步：若有條目，直接引用 vram_min/vram_rec/ram_min/ram_rec/disk。若無條目，上 Hugging Face 或 GitHub 找官方推論硬體需求（如官方宣稱「1024×1024 FP16 需 3.45GB VRAM」），或參照同類型模型推斷。第三步：確實填寫，不得寫「雲端服務」「無需特定」這類廢話。
    "vram_min": （數字，最低 GPU VRAM 需求，單位 GB），
    "vram_rec": （數字，建議 GPU VRAM，單位 GB），
    "ram_min": （數字，最低系統 RAM 需求，單位 GB），
    "ram_rec": （數字，建議系統 RAM，單位 GB），
    "disk": （數字，硬碟空間需求，單位 GB），
    "cpu": （字串，描述 CPU 最低規格），
    "power": （字串，描述電源供應需求，依 GPU 而定）
